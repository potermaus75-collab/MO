# 몬스터 육성 (웹/HTML/CSS/JS 프로토타입)

이 프로젝트는 **정적(Static) 웹**으로 동작하는 "몬스터 육성" 중심 프로토타입입니다.
탐험/조우/배틀/포획은 유지하되, 핵심 루프는 **내 땅(베이스) → 서식지 배치 → 먹이/훈련 → 진화 → 성체부터 교배** 입니다.
데이터는 기존 SQLite(DB v3)에서 **웹에서 쓰기 좋은 JSON**으로 내보낸 것을 사용합니다.

## 실행 방법 (로컬)

브라우저에서 `file://`로 바로 열면 `fetch()`가 막힙니다. **로컬 서버로 실행**하세요.

### Python이 있는 경우
```bash
cd .
python -m http.server 8000
```
그 후 브라우저에서 `http://localhost:8000` 접속.

### Node가 있는 경우
```bash
npx serve .
```

## GitHub Pages 배포

1. 이 폴더 전체를 GitHub 리포지토리에 업로드

3. Branch를 `main` / folder `/ (root)` 로 선택
4. 배포 URL로 접속

## 조작법

- 이동: 방향키 / WASD
- 상호작용: Enter
- 메뉴: M (또는 화면 우측 상단 메뉴 버튼)
- 저장: 우측 상단 저장 버튼
- 모드 전환: 우측 상단 **내 땅 / 탐험** 버튼

## 게임 루프

### 내 땅(육성)

- **서식지 건설**: 속성별 서식지를 만들고 몬스터를 배치
- **터치(클릭) 육성**: 서식지 안의 몬스터를 클릭 → 먹이/훈련/진화/이동
- **교배 동굴**: 성체(이상) + 암/수 페어를 넣고 시간이 지나면 알 생성 → 부화

> 시간은 데모용으로 빠르게 흘러갑니다. `data/game/config.json`의 `time.minuteMs`를 `60000`으로 바꾸면 1분=60초(현실 시간) 기준으로 동작합니다.

### 탐험

- **수풀(초록색) 위를 걸으면** 랜덤 조우(야생)
- **나무(T), 물(W), 용암(L)** 은 **인접한 상태에서 Enter로 조사** → 랜덤 조우
- NPC에게 Enter로 대화/배틀
- 야생 배틀에서 포획구로 포획

## 데이터 구조

- `data/master/` : 몬스터/스킬/상태이상/진화 등(기존 DB에서 export)
- `data/world/` : 지방/맵/스폰테이블/NPC (웹용 게임 컨텐츠)
- `data/game/` : 런타임 아이템/레벨업 곡선/게임 설정
